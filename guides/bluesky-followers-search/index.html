Bluesky Followers Search

In the official Bluesky app, searching followers of one specific profile is limited. GlimpSky provides a dedicated searchable follower/following modal.

Open Viewer Try Example Profile

How to search followers or following

  1. Load a profile in GlimpSky.
  2. Click the followers or following stat on the profile card.
  3. In the list modal, search by name, @handle, or /regex/flags.
  4. If Search all appears, click it (or press Enter) to search the full list, not only loaded entries.
  5. Click any result to open that account in GlimpSky.

Regex is useful when plain text is too strict, for example matching multiple name variants with one query. Guide: MDN Regular expressions.

Also available in this modal workflow

  • Mutuals list with cached computation.
  • Blocking list from public records.
  • Load more pagination for large lists.

Limitations

  • Very large lists can take time to fully load for complete search.
  • Mutuals can be partial on very large accounts (current computation cap).
  • Blocked by is currently unavailable without an external indexer.
