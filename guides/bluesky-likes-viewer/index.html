Bluesky Likes Viewer

Analyze what public Bluesky accounts like, when they like it, and who they engage with using advanced filters.

Open likes view Try example profile

How to see likes (quick steps)

  1. Open GlimpSky and enter a public handle or DID.
  2. Select View likes.
  3. Apply filters (date, text, author, quotes, links, oldest-first).
  4. If prompted, click Load and apply and wait for full loading.

Filters supported in Likes view

  • Date range: filters by like timestamp.
  • Search posts by text: plain text or /regex/flags.
  • Search by author: handle, DID, or display name; plain text or regex.
  • Hide quotes, Only links, and Oldest first.

Regex is better than plain text when you need flexible matching (alternatives, partial patterns, and flags like case-insensitive search). Guide: MDN Regular expressions.

Likes analytics (likes-specific)

  • Timeline includes one aggregate All likes line plus category lines (text, links, media, replies, quotes).
  • Use Show only All likes line to hide category lines when you want one clean trend line.
  • Content mix and Top liked authors update with the selected analytics range.
  • Click timeline points/buckets to apply date filters, and click top liked authors to apply author filters.

For shared analytics behavior (range buttons, custom/date-filter behavior, loading coverage), see Profile analytics workflow.

Find likes tied to another profile

The official Bluesky app does not provide direct two-profile interaction search. In GlimpSky you can do this with author filtering.

  1. Enter profile A in the main handle field.
  2. Enter profile B in Search by author.
  3. Use plain text or regex and run Load and apply for complete results.

Counts and known limits

  • Likes stat may start as a partial value (for example 2000+); click it to compute exact likes.
  • If coverage is partial, use Load more or Load all data to include older likes.
  • Some likes may be unavailable if source records/posts are deleted, deactivated, or temporarily unreachable.

Author matching can include post author, reply parent author, and quoted author, which is useful for interaction analysis.
