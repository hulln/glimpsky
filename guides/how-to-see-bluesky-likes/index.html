How to See Bluesky Likes

This topic is now maintained as a subsection of the main Likes guide to avoid duplicated documentation.

Open the Likes How-To Section Open Likes View

Quick answer

  1. Enter a public handle or DID.
  2. Click View Likes.
  3. Apply filters and use Load and Apply when prompted.

For full details (regex, author search, only-links, oldest-first, exact likes recount), use the main Likes guide linked above. Regex reference: MDN Regular expressions.
