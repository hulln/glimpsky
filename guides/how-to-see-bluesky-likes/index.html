How to See Bluesky Likes

A practical walkthrough for viewing likes on public Bluesky profiles using GlimpSky.

Open Likes View Try With Example

Step by step

  1. Open GlimpSky.
  2. Paste a public Bluesky handle (like user.bsky.social) or DID.
  3. Click View Likes.
  4. Use filters: date range, search text, author search, hide quotes, oldest-first.
  5. Click any result to open the source post on Bluesky.

Useful filter combinations

Topic Exploration

Use text search with keywords like atproto or privacy.

Time Window

Set date range to inspect activity during a specific event period.

Author Pattern

Use author search (plain text or regex) to find recurring liked accounts.

Oldest-First Review

Load all data, then sort oldest-first to understand changes over time.

If you need complete results on large accounts, use "Load and Apply" when prompted by advanced filters.
