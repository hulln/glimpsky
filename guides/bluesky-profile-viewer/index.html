Bluesky Profile Viewer

Use GlimpSky to analyze public Bluesky accounts beyond the default app view: content, social graph, and account diagnostics in one workflow.

Open Viewer Try Example Profile

Quick start

  1. Enter a handle or DID (for example, user.bsky.social).
  2. Choose View Posts or View Likes.
  3. Use filters (text, author, date, quotes, links, oldest-first).
  4. If prompted, click Load and Apply to load full data for advanced filters.

Profile diagnostics (not shown in the Bluesky app)

  • Joined date, account age, last active, and last follow date.
  • Identity update date and Home PDS.
  • Last 30 days counters: posts, likes, follows, reposts.

Social graph tools

  • Clickable followers and following stats open searchable lists.
  • List search supports plain text and /regex/flags, with Search all for full-list scan.
  • Mutuals and blocking are computed from public data.
  • Blocked by is shown as unavailable (requires external indexing).

Content analysis features

  • Mode switch: View Posts and View Likes.
  • Filters: hide reposts, hide replies, hide quotes, only links, oldest-first, date range, text search, author search.
  • Text and author search both support plain text and regex.
  • Likes count can show partial totals first; click likes stat for an exact recount.

Regex helps you match variable patterns (for example alternatives, partial matches, and case-insensitive search) that plain text cannot. Guide: MDN Regular expressions.

Workflow helpers

  • Handle input supports typeahead suggestions.
  • Load More pagination is available before full-load mode.
  • Each item can be opened on Bluesky, and post links can be copied directly.

Find interactions between two profiles

Bluesky does not offer direct "profile A with profile B" interaction search. GlimpSky can do this with filters.

  1. Enter profile A in the main handle field.
  2. In Search by author, enter profile B (handle, DID, or display name).
  3. Use plain text or regex, then click Load and Apply and wait for completion.

Tip: author search can match post authors, reply parent authors, and quoted authors, so it is useful for relationship analysis.
